# Pichau UI

## Tecnologias Utilizadas

- **Vue.js**: ^3.4.37
- **Pinia**: ^2.2.5
- **Vue Router**: ^4.4.5
- **Vite**: ^5.4.1
- **ESLint**: ^8.57.1
- **Prettier**: ^3.0.0

## Como Rodar a Aplicação

### Pré-requisitos

- Node.js instalado (recomendado: versão LTS)
- npm ou yarn instalado

### Passos para Rodar

1. Clone o repositório:

    ```sh
    cd pichau-ui
    ```

2. Instale as dependências:

    ```sh
    npm install
    # ou
    yarn install
    ```

3. Rode a aplicação em modo de desenvolvimento:

    ```sh
    npm run dev
    # ou
    yarn dev
    ```

4. Acesse a aplicação no navegador:

    Abra o navegador e vá para `http://localhost:3000`

### Outros Comandos Úteis

- **Build**: Para criar uma versão de produção da aplicação:

    ```sh
    npm run build
    # ou
    yarn build
    ```

- **Preview**: Para visualizar a versão de produção localmente:

    ```sh
    npm run preview
    # ou
    yarn preview
    ```

- **Lint**: Para verificar a qualidade do código:

    ```sh
    npm run lint
    # ou
    yarn lint
    ```

- **Format**: Para formatar o código:

    ```sh
    npm run format
    # ou
    yarn format
    ```
